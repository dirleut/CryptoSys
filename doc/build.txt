---Сборка Windows---
1) Скачать Visual Studio 2019 Community  (можно и 2015 или 2017)
2) При конфигурации выбрать компонент "Разработка классических приложений на C++"
3) Скачать JUCE с официальной страницы https://shop.juce.com/get-juce/download
4) Распаковать и запустить Projucer
5) Зарегиститроваться и создать пустой проект (Create Blank Project)
	В открытом окне придумать название проекту и в качестве экспорта (Target Platforms)
	выбрать свою версию Visual Studio. Создастся и откроется одноимённое .jucer-приложение, через которое 
	добавляются файлы, задаются параметры сборки и тд. Удалить там папку Source.
	
6) Скачать из репозитория исходные файлы проекта https://github.com/dirleut/CryptoSys (Download ZIP, например)
7) Добавить исходные файлы (обязательно) через .jucer-приложение. Нажать на + "Add Existing Files".
	(Пока необязательно папки добавлять через "Add New Group").

8) Экспортировать в Visual Studio
9) В открытом проекте Visual Studio нажать "Сборка" -> "Собрать решение"
10) Запустить через "Отладка" -> "Начать отладку" (или "Запуск без отладки").
	Можно также порыться в папке Builds и найти сам .exe-файл

---Создание GUI-элементов с автоматической генерацией кода---
1) Открыть .jucer-приложение нужного проекта
2) Нажать на + "Add New GUI Component" и придумать название
3) Кликнуть на сгенерировавшийся .cpp файл
4) На открывшемся поле добавить необходимые кнопки, текст и тд.
	Правая кнопка мыши и "New ..." Это автоматически сгенерирует нужный код.
