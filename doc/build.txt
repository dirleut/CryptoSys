Сборка учебного комплекса:

-Windows
	1) Скачать Visual Studio 2019 Community  (можно и 2015 или 2017)
	2) При конфигурации выбрать компонент "Разработка классических приложений на C++"
	3) Любым способом скачать из репозитория исходные файлы https://github.com/dirleut/CryptoSys
	4) Скачать JUCE с официальной страницы https://shop.juce.com/get-juce/download
	5) Распаковать и запустить Projucer
	6) Зарегеститроваться и создать пустой проект. В качестве экспорта проекта выбрать свою версию Visual Studio. Создастся одноимённое .jucer-приложение, через которое добавляются файлы, задаются параметры сборки и тд.
	7) Добавить исходные файлы (обязательно) через .jucer
	8) Экспортировать в Visual Studio
	9) Собрать!

-Mac OS
	То же самое, но с Xcode
